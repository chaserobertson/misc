The Data Set

Stats NZ publishes a Overseas merchandise trade dataset which records all imports and exports to/from Aotearoa New Zealand on a monthly basis (see “Notes” on the bottom of the website for variable descriptions).

For this lab we will use a small subset of the “Imports” dataset which has all imports of goods falling into the Harmonised Systems (HS) code 8703231915 which is defined as “Vehicles; with only spark-ignition internal combustion reciprocating piston engine, cylinder capacity over 1500 but not over 3000cc, new motor cars, assembled” in the csv files called car-imports.csv. You can find the file on the VMs in the directory /course/data/trade or on Canvas.

   Month,Unit Qty,Country,vfd,cif,Imports Qty
   201901,"NMB","Austria","2,511,892","2,574,059","42"
   201901,"NMB","Belgium","1,886,314","1,963,103","53"
   201901,"NMB","China","2,675,272","2,799,368","121"
   201901,"NMB","Czechia","7,251,350","7,665,974","256"
   201901,"NMB","Germany","16,902,288","17,320,488","308"

Every line represents the imports of the above mentioned cars from a particular country in a particular month. For simplicity we will refer to this data as “car imports” even though it is only a subset of all imported cars. (For a real analysis we would combine all HS codes related to cars, but that would go beyond the scope of this lab.)
